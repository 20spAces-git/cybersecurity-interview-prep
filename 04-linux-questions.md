# Linux Interview Questions

## What is Linux?

Answer:

Linux is an open-source operating system commonly used in servers, cybersecurity environments, cloud infrastructure, and enterprise systems.

Many cybersecurity tools and platforms run on Linux.

---

## Why is Linux Important in Cybersecurity?

Answer:

Linux is widely used because it is:

- Secure
- Stable
- Highly configurable
- Common in enterprise environments

Cybersecurity professionals often use Linux for:

- Incident Response
- Digital Forensics
- Penetration Testing
- Threat Hunting
- System Administration

---

## What Does pwd Do?

Answer:

pwd stands for Print Working Directory.

It displays the current location within the Linux file system.

Example:

pwd

---

## What Does ls Do?

Answer:

ls lists files and directories within the current location.

Examples:

ls

ls -la

---

## What Does cd Do?

Answer:

cd stands for Change Directory.

It allows users to move between directories.

Example:

cd /home/user

---

## What Does mkdir Do?

Answer:

mkdir creates a new directory.

Example:

mkdir cyberlab

---

## What Does touch Do?

Answer:

touch creates a new empty file.

Example:

touch notes.txt

---

## What Does cp Do?

Answer:

cp copies files or directories.

Example:

cp file1.txt file2.txt

---

## What Does mv Do?

Answer:

mv moves or renames files and directories.

Example:

mv old.txt new.txt

---

## What Does rm Do?

Answer:

rm removes files and directories.

Example:

rm file.txt

Important:

Use caution because deleted files are not automatically recoverable.

---

## What Does chmod Do?

Answer:

chmod changes file permissions.

Example:

chmod 755 file.txt

---

## What Does chown Do?

Answer:

chown changes file ownership.

Example:

chown user:user file.txt

---

## What Does sudo Do?

Answer:

sudo allows authorized users to execute commands with elevated privileges.

Example:

sudo apt update

---

## What Command Shows Running Processes?

Answer:

ps

Example:

ps aux

This displays:

- Running Processes
- Process IDs (PID)
- Resource Usage
- Users

---

## What Does top Do?

Answer:

top displays real-time system activity.

It allows administrators and analysts to monitor:

- CPU Usage
- Memory Usage
- Running Processes

---

## What Are Linux Log Files?

Answer:

Log files record system activity and events.

Common Locations:

/var/log

Examples:

- auth.log
- syslog
- kern.log

---

## Why Are Logs Important?

Answer:

Logs help investigators:

- Identify suspicious activity
- Review login attempts
- Build timelines
- Investigate incidents

Logs are one of the most important sources of evidence during DFIR investigations.

---

## What Is the Difference Between a User and a Group?

Answer:

A user is an individual account.

A group is a collection of users that share permissions and access rights.
